# ⚡ FooRniz optimizer BETA

**FooRniz optimizer BETA** — программа для очистки, оптимизации Windows и запуска встроенного Zapret.

## 🚀 Возможности

- 🧹 Быстрая и глубокая очистка системы
- 🧠 Оптимизация ОЗУ
- 🎮 Твики для игр и производительности
- 💾 TRIM/оптимизация SSD и HDD
- 🛠️ Проверка Windows через SFC, DISM и CHKDSK
- 🌍 Встроенный Zapret для обхода Roblox, YouTube, Discord и многого другого
- 🔁 Автозапуск Zapret при входе в Windows
- 📋 Простое меню на русском языке

## 📌 Запуск

1. Скачай проект.
2. Запусти `optimize.bat`.
3. Подтверди запуск от имени администратора.
4. Выбери нужный пункт меню.

## 🌍 Zapret

В программе есть встроенный запуск Zapret.

После запуска Zapret окно оптимизатора можно закрыть — Zapret продолжит работать отдельно.

Также можно включить автозапуск Zapret, чтобы он запускался сам при входе в Windows.

## ⚠️ Важно

- Запускай программу от имени администратора.
- Перед серьезными твиками рекомендуется создать точку восстановления.
- Программа не удаляет личные файлы из Документов, Рабочего стола и Загрузок.
- Если не уверен, сначала используй быструю очистку.
